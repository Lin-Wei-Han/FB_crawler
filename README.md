## Clone this Repository

1. 打開 Terminal

```bash
git clone https://github.com/Lin-Wei-Han/FB_crawler.git
```

2. 進入專案資料夾

```bash
cd FB_crawler
```

3. 建立 Python 虛擬環境（windows）

```bash
python -m venv venv
```

4. 進入 Python 虛擬環境

```bash
.\venv\Scripts\activate
```

5. 安裝套件
   requirements.txt 用於紀錄專案使用到的套件及版本

```bash
pip install -r requirements.txt
```
